# QM_sim

Python library for simulation of quantum mechanical systems.

## Features 
- 1D and 2D systems
- Choice of finite difference scheme
- Stationary solutions

## Planned features
- Boundary conditions
- 3D systems
- Temporal simulation
- Time-variant potentials
- Example plots

## Installation

`pip install qm-sim`

## Usage

Examples are provided in the `examples/`-folder
